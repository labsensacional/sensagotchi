## 1. Fix estimulantes: sleepiness↓ en vez de energy↑, agregar hunger↓

Los estimulantes no dan energía real — bloquean adenosina (sueño) y suprimen apetito.

**`events.js`** — funciones a modificar:

```js
// cocaine (~línea 1373):
// ANTES: h.energy += 15;
// DESPUÉS:
h.sleepiness -= 30;   // adenosine blockade
h.hunger -= 18;       // appetite suppression

// amphetamines (~línea 1344): ya tiene h.hunger -= 20
// ANTES: h.energy += 20;
// DESPUÉS:
h.sleepiness -= 40;

// caffeine (~línea 1299): ya tiene h.sleepiness -= 25 * eff
// ANTES: h.energy += 10;
// DESPUÉS: simplemente eliminar esa línea (o dejar un +3 simbólico)
// también agregar: h.hunger -= 8;  (caffeine suprime apetito)

// tobacco (~línea 1281):
// ANTES: h.energy += 3;
// DESPUÉS:
h.sleepiness -= 5;
h.hunger -= 3;
```

También actualizar los `can_apply` que dicen `'need more energy'` — deberían permitirse con energía baja si sleepiness es alta (la gente toma cocaína justamente cuando está cansada). Cambiar condición de cocaine/amphetamines de `h.energy > 20` a `h.energy > 5 || h.sleepiness > 40`.

## 2. Nueva variable: `anandamide` (sistema endocannabinoide)

Anandamida = "molécula de la bienaventuranza". Sistema tónico (lento), modula mood de fondo, dolor, apetito. Cannabis actúa sobre los mismos receptores CB1.

### En `human.js`:

```js
// Agregar a _FLOAT_FIELDS:
'anandamide',

// Agregar a _CLAMPED_FLOAT_FIELDS:
'anandamide',

// En constructor, agregar parámetro con default:
anandamide = 30.0,

// Y en el body:
this.anandamide = anandamide;  // endocannabinoid tone — background bliss, pain mod, appetite
```

### En `events.js`:

```js
// BASELINES — agregar:
anandamide: 30.0,

// DECAY_RATES — agregar (más lento que dopamina, sistema tónico):
anandamide: 0.08,
```

### Acciones que boostean anandamide (`events.js`):

```js
// exercise: runner's high es endocannabinoide, no endorfinas
h.anandamide += 18 * eff;   // agregar en function exercise()

// orgasm:

// cuddling:
```

## 3. Nuevo método `emotional_tone_score()` en `human.js`

Tono emocional = piso estable del estado de ánimo. Cambia lento, se setea por presets.

```js
emotional_tone_score() {
  /**
   * Background emotional tone — the stable mood floor.
   * Driven by slow-moving variables: serotonin (contentment),
   * oxytocin (warmth/safety), anandamide (endocannabinoid bliss).
   * Range: 0–100.
   */
  return (
    this.serotonin              * 0.35 +
    this.oxytocin               * 0.30 +
    (this.anandamide ?? 30)     * 0.35
  );
}
```

## 4. Restructurar `liking_score()` en `human.js`

**Concepto**: endorfinas y dopamina son los picos dinámicos (lo que generan las acciones). Serotonina/oxitocina/anandamida son el piso/tono que amplifica o amortigua esos picos.

```js
liking_score() {
  // Dynamic hedonic peaks — generated by actions
  const dynamic_peaks = this.endorphins * 0.65 + this.dopamine * 0.12;

  // Emotional floor — slow-moving tone acts as context/amplifier
  const emotional_tone = this.emotional_tone_score();
  
  // Base liking: peaks + tone floor (additive — tone adds a stable background)
  const base_liking = dynamic_peaks + emotional_tone * 0.30;
  // At baseline (endorphins=20,dopamine=50,serotonin=50,oxytocin=30,anandamide=30):
  //   peaks=13+6=19, tone=17.5+9+10.5=37, base=19+11.1=30.1  ← similar al ~33 anterior

  // Yerkes-Dodson (sin cambio):
  const optimum = this.yerkes_dodson_optimum();
  let anxiety_factor;
  if (this.anxiety <= optimum) {
    anxiety_factor = 0.92 + (this.anxiety / optimum) * 0.13;
  } else {
    anxiety_factor = 1.05 - ((this.anxiety - optimum) / (100 - optimum)) * 0.45;
  }

  // Absorption bonus (sin cambio):
  const ssri_pct = this.ssri_level / 100.0;
  const max_bonus = 0.3 * (1 - ssri_pct * 0.5);
  const absorption_factor = 1.0 + (this.absorption / 100) * max_bonus;

  // Shutdown (sin cambio):
  const shutdown_factor = 1.0 - (this.shutdown / 100) * 0.8;

  return base_liking * anxiety_factor * absorption_factor * shutdown_factor;
}
```

## 5. Restructurar `wanting_score()` en `human.js`

**Concepto**: dopamina es la señal pura de deseo. Vasopresina modula la intensidad del drive (comportamiento de búsqueda/persecución).

```js
wanting_score() {
  // Dopamine is the pure wanting signal; arousal adds urgency
  const dynamic_want = this.dopamine * 0.55 + this.arousal * 0.25;

  // Vasopressin amplifies drive intensity (goal-directed pursuit behavior)
  const vaso_drive = 0.70 + (this.vasopressin / 100) * 0.55;  // 0.70–1.25

  let base_wanting = dynamic_want * vaso_drive;
  // At baseline (dopamine=50, arousal=20, vasopressin=20):
  //   dynamic=27.5+5=32.5, vaso=0.70+0.11=0.81, base=26.3  ← similar al ~30 anterior

  // Cue salience (sin cambio):
  const salience_values = Object.values(this.cue_salience);
  const max_salience = salience_values.length > 0 ? Math.max(...salience_values) : 0;
  base_wanting += max_salience * 25;

  // Prolactin suppression (sin cambio):
  const prolactin_factor = 1.0 - (this.prolactin / 100) * 0.5;

  // Low energy suppression (sin cambio):
  const energy_factor = 0.6 + (this.energy / 100) * 0.4;

  // Shutdown collapse (sin cambio):
  const shutdown_factor = 1.0 - (this.shutdown / 100) * 0.6;

  return base_wanting * prolactin_factor * energy_factor * shutdown_factor;
}
```
