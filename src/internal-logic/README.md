# internal-logic-js

JavaScript port of the sensagotchi simulation engine.

Requires **Node 18+**. If you're on an older version, use nvm:
```
nvm use 20
```

## Setup
```
npm install
```

## Run tests
```
npm test
```

68 tests covering all axioms (neuroscience-based behavioral invariants), opponent-process theory, drug events, and probabilistic outcomes.
